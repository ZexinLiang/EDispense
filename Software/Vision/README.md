# Vision / 视觉软件

[简体中文](#简体中文) | [English](#english)

## 简体中文

### 简介
本目录包含 EDispense 系统的**机器视觉**相关内容，用于 PCB 检测、焊盘识别，以及点胶过程中的缺陷检测。

### 内容
- **Pad Detection（焊盘检测）**：PCB 焊盘检测与定位算法。
- **Calibration（标定）**：确保视觉系统与点胶机构准确对齐的标定流程。
- **Inspection（检测）**：对焊膏沉积进行检查，识别少锡、过量、偏移等缺陷。
- **Datasets（数据集）**：用于训练与测试视觉算法的图像数据集。

### 目的
本目录用于管理系统的视觉模块，通过正确识别并验证焊膏沉积的位置与质量，提升点胶过程的精度与可靠性。

## English

### Introduction
This directory contains all files related to the **machine vision** system used for PCB inspection, solder pad recognition, and defect detection during the dispensing process.

### Contents
- **Pad Detection**: Algorithms for detecting and localizing solder pads on the PCB.
- **Calibration**: Calibration routines to ensure the vision system is accurately aligned with the dispensing mechanism.
- **Inspection**: Software for inspecting the solder deposit, checking for defects like insufficient solder, excess solder, or misalignment.
- **Datasets**: Image datasets for training and testing the vision algorithms.

### Purpose
The purpose of this directory is to manage the vision-based components of the system, ensuring that the dispensing process is precise and reliable by correctly identifying and verifying the position and quality of solder deposits.

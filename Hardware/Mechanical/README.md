# Mechanical / 机械设计

[简体中文](#简体中文) | [English](#english)

## 简体中文

### 简介
本目录包含 EDispense 系统的**机械设计**相关文件，包括机架、运动机构、点胶头以及相关机械装配件。

### 内容
- **Frame（机架）**：结构件与外壳/机架设计文件。
- **Motion（运动系统）**：执行器、直线导轨与运动机构相关设计。
- **Dispenser（点胶机构）**：点胶头设计（针头、出胶/挤出机构及相关部件）。
- **Fixtures（工装夹具）**：用于点胶过程中定位 PCB 的夹具、支架与固定件。

### 目的
本目录用于提供 EDispense 系统稳定、精确运行所需的实体结构与机械部件。

## English

### Introduction
This directory contains all the files related to the **mechanical design** of the EDispense system, including the frame, motion components, dispensing head, and related mechanical assemblies.

### Contents
- **Frame**: Structural components and design files for the physical enclosure.
- **Motion**: Designs for the motion system, including actuators, linear guides, and motion control.
- **Dispenser**: The design of the dispensing head, including the needle, dispensing mechanism, and related components.
- **Fixtures**: Any custom fixtures, mounts, or holders used to position the PCB during dispensing.

### Purpose
The purpose of this directory is to provide the physical structure and mechanical components needed for the accurate and stable operation of the EDispense system.
